# Jogos Desenvolvidos com PICO-8

Este junta pequenos jogos que desenvolvi utilizando o **PICO-8**, focados em mecânicas simples e jogos curtos.

## Lista de Jogos

* **Jogo da Fazenda — Versão PC**
  Desenvolvido para computador, com controles baseados principalmente no movimento do mouse.

* **Jogo da Fazenda — Versão Mobile / Gamepad**
  Adaptado para dispositivos móveis e controle via gamepad, utilizando as setas direcionais e botões específicos (como **X** e **Z**).
